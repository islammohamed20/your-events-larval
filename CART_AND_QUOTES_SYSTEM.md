# 🛒 نظام سلة التسوق و عروض الأسعار - Your Events

**التاريخ:** 11 أكتوبر 2025  
**الإصدار:** 2.0

---

## 📋 نظرة عامة

تم إضافة نظام كامل لسلة التسوق (Shopping Cart) وعروض الأسعار (Quotes) إلى موقع Your Events مع الميزات التالية:

### ✨ الميزات الجديدة:

1. **🛒 سلة تسوق كاملة**
   - إضافة خدمات للسلة
   - تعديل الكميات
   - إضافة ملاحظات لكل خدمة
   - حساب تلقائي للضرائب (15%)

2. **💰 نظام عروض الأسعار**
   - تحويل السلة إلى عرض سعر
   - متابعة حالة العروض
   - تحميل PDF
   - إضافة ملاحظات

3. **🎯 إخفاء ذكي**
   - إخفاء صفحة الباقات عند عدم وجود باقات
   - إخفاء صفحة المعرض عند عدم وجود محتوى

4. **👨‍💼 لوحة تحكم للإدارة**
   - مراجعة عروض الأسعار
   - الموافقة/الرفض
   - إضافة خصومات
   - إضافة ملاحظات للعميل

## 🗄️ قاعدة البيانات

### جداول جديدة:

#### 1. جدول `quotes` (عروض الأسعار)
```sql
- id: المعرف الفريد
- user_id: معرف العميل
- quote_number: رقم العرض (QT-20250001)
- status: الحالة (pending, approved, rejected, completed)
- subtotal: المجموع الفرعي
- tax: الضريبة (15%)
- discount: الخصم
- total: الإجمالي
- customer_notes: ملاحظات العميل
- admin_notes: ملاحظات الإدارة
- approved_at: تاريخ الموافقة
- rejected_at: تاريخ الرفض
- created_at, updated_at
```

#### 2. جدول `quote_items` (تفاصيل عروض الأسعار)
```sql
- id: المعرف الفريد
- quote_id: معرف العرض
- service_id: معرف الخدمة
- service_name: اسم الخدمة
- service_description: وصف الخدمة
- price: السعر
- quantity: الكمية
- subtotal: المجموع الجزئي
- customer_notes: ملاحظات خاصة بالخدمة
- created_at, updated_at
```

#### 3. جدول `cart_items` (سلة التسوق)
```sql
- id: المعرف الفريد
- user_id: معرف المستخدم المسجل
- session_id: معرف الجلسة للزوار
- service_id: معرف الخدمة
- quantity: الكمية
- price: السعر
- customer_notes: ملاحظات العميل
- created_at, updated_at
```

## 🎨 واجهات المستخدم

### للعملاء:

#### 1. صفحة السلة (`/cart`)
- عرض جميع الخدمات في السلة
- تعديل الكميات
- حذف خدمات
- عرض الملاحظات
- ملخص الأسعار مع الضريبة
- زر إنشاء عرض سعر

#### 2. صفحة عروض الأسعار (`/quotes`)
- عرض جميع عروض الأسعار
- فلترة حسب الحالة
- عرض تفاصيل كل عرض
- تحميل PDF
- تعديل الملاحظات (قبل الموافقة)

#### 3. صفحة عرض سعر واحد (`/quotes/{id}`)
- تفاصيل الخدمات
- ملخص الأسعار
- حالة العرض
- ملاحظات العميل والإدارة
- زر تحميل PDF
- زر تأكيد الحجز (إذا تمت الموافقة)

#### 4. صفحة الخدمة (محدثة)
- نموذج إضافة للسلة
- حقل الكمية
- حقل الملاحظات
- زر إضافة للسلة
- زر حجز مباشر (القديم)

### للإدارة:

#### لوحة تحكم عروض الأسعار (`/admin/quotes`)
- عرض جميع العروض
- إحصائيات (إجمالي، قيد الانتظار، موافق، مرفوض، مكتمل)
- بحث بالرقم أو العميل
- فلترة حسب الحالة
- عرض تفاصيل
- تغيير الحالة
- إضافة خصم
- إضافة ملاحظات
- حذف

## 🔄 سير العمل (Workflow)

### 1. العميل يضيف خدمات:
```
صفحة الخدمة → إضافة للسلة (مع ملاحظات) → السلة
```

### 2. إنشاء عرض سعر:
```
السلة → إضافة ملاحظات عامة → Checkout → إنشاء عرض سعر
```

### 3. مراجعة الإدارة:
```
لوحة التحكم → عروض الأسعار → مراجعة → الموافقة/الرفض + ملاحظات
```

### 4. العميل يراجع:
```
عروض الأسعار → عرض التفاصيل → تحميل PDF → تأكيد الحجز
```

## 🚀 Routes الجديدة

### للعملاء:

```php
// Cart
GET  /cart                    - عرض السلة
POST /cart/add/{service}      - إضافة خدمة
PATCH /cart/{cartItem}        - تحديث كمية
DELETE /cart/{cartItem}       - حذف خدمة
POST /cart/clear              - تفريغ السلة
GET /cart/count               - عدد العناصر (AJAX)

// Quotes
GET  /quotes                  - عروض الأسعار
GET  /quotes/{quote}          - عرض تفاصيل
POST /quotes/checkout         - إنشاء عرض من السلة
GET  /quotes/{quote}/download - تحميل PDF
PATCH /quotes/{quote}/notes   - تحديث ملاحظات
```

### للإدارة:

```php
// Admin Quotes
GET    /admin/quotes                - جميع العروض
GET    /admin/quotes/{quote}        - التفاصيل
PATCH  /admin/quotes/{quote}/status - تحديث الحالة
DELETE /admin/quotes/{quote}        - حذف
```

## 💡 الميزات التقنية

### 1. حساب تلقائي للأسعار:
```php
$subtotal = items->sum('subtotal');
$tax = $subtotal * 0.15;  // 15% VAT
$total = $subtotal + $tax - $discount;
```

### 2. أرقام عروض أسعار فريدة:
```
Format: QT-{YEAR}{NUMBER}
Example: QT-20250001, QT-20250002
```

### 3. دعم الزوار غير المسجلين:
- السلة تعمل بـ `session_id`
- عند تسجيل الدخول، يمكن ربط السلة بالمستخدم

## 🐛 حل المشاكل

### المشكلة: السلة فارغة بعد إضافة خدمة
**الحل:**
- تأكد من تسجيل الدخول أو أن Session نشط
- تحقق من Console للأخطاء

### المشكلة: PDF لا يعمل
**الحل:**
```bash
composer require barryvdh/laravel-dompdf
php artisan config:clear
```

### المشكلة: عدد السلة لا يتحدث
**الحل:**
- تحقق من `cart-count` في الـ blade
- تأكد من AJAX يعمل
- F12 → Network → تحقق من الطلبات
