```

{{website_url}}      - رابط الموقع

```
{{customer_name}}    - اسم العميل

{{expiry_time}}      - مدة الصلاحية

```

{{invoice_date}}     - تاريخ الفاتورة

## ✨ المميزات الإضافية

### 1️⃣ **معاينة القالب**
- اضغط على أيقونة العين 👁️
- شاهد كيف سيبدو البريد للعملاء
- بيانات تجريبية للمتغيرات

```
1. افتح القالب للتعديل أو المعاينة
2. في الشريط الجانبي → "إرسال بريد تجريبي"
3. أدخل بريدك الإلكتروني
4. اضغط "إرسال الآن"
5. تحقق من البريد الوارد

- اضغط على أيقونة النسخ 📋
- يتم إنشاء نسخة من القالب
- عدّل النسخة كما تشاء

- استخدم الزر في قائمة القوالب
- القوالب المعطلة لن تُرسل تلقائياً

## 🔄 استخدام القوالب في الكود

### مثال: إرسال بريد عند الحجز

```php
use App\Models\EmailTemplate;
use Illuminate\Support\Facades\Mail;

// الحصول على القالب
$template = EmailTemplate::where('slug', 'booking-confirmation')
    ->where('is_active', true)
    ->first();

if ($template) {
    // البيانات المطلوبة
    $data = [
        'customer_name' => $booking->user->name,
        'booking_number' => $booking->id,
        'service_name' => $booking->service->name,
        'booking_date' => $booking->booking_date->format('Y/m/d'),
        'booking_time' => $booking->booking_time,
        'total_amount' => number_format($booking->total_amount, 2),
        'company_name' => 'Your Events',
        'company_phone' => '0500000000',
        'company_email' => 'info@yourevents.sa',
    ];

    // استبدال المتغيرات
    $rendered = $template->render($data);

    // إرسال البريد
    Mail::send([], [], function ($message) use ($booking, $rendered) {
        $message->to($booking->user->email)
            ->subject($rendered['subject'])
            ->html($rendered['body']);
    });
}
```

## ⚡ نصائح مهمة

### ✅ افعل:
1. **اختبر القوالب** قبل تفعيلها
2. **استخدم المتغيرات** لجعل المحتوى ديناميكي
3. **احفظ نسخة احتياطية** من القوالب المهمة
4. **راجع التنسيق** في المعاينة
5. **استخدم HTML بسيط** لضمان التوافقية

### ❌ لا تفعل:
1. لا تحذف القوالب المستخدمة في الكود
2. لا تستخدم JavaScript (لن يعمل في الإيميلات)
3. لا تنسى تفعيل القالب بعد التعديل
4. لا تستخدم صور خارجية بدون https
5. لا تترك المتغيرات بدون قيمة في الكود

## 🛠️ استكشاف الأخطاء

### ❌ "البريد لم يُرسل"
**الحل:**
1. تأكد من تفعيل القالب
2. راجع إعدادات البريد في `.env`
3. اختبر من صفحة "اختبار البريد"

### ❌ "المتغيرات لا تُستبدل"
**الحل:**
1. تأكد من كتابة المتغير بشكل صحيح: `{{variable}}`
2. تأكد من تمرير القيمة في الكود
3. راجع المعاينة للتأكد

### ❌ "التنسيق غير صحيح"
**الحل:**
1. استخدم محرر TinyMCE
2. راجع المعاينة
3. اختبر على بريد حقيقي
