# 📘 دليل إضافة الخيارات/القيم للخصائص

## 🎯 الهدف
إضافة القيم المختلفة لكل خاصية (مثل: إضافة "50-100"، "100-200"، "200-300" لخاصية "عدد الضيوف")

---

## 📋 الطريقة الأولى: من صفحة إدارة الخصائص (الأسهل)

### الخطوة 1️⃣: الدخول إلى إدارة الخصائص
```
1. افتح لوحة التحكم
2. من القائمة الجانبية → "خصائص الخدمات"
3. أو اذهب مباشرة إلى: https://yourevents.sa/admin/attributes
```

### الخطوة 2️⃣: اختيار الخاصية
```
1. ستجد قائمة بجميع الخصائص
2. اضغط على زر "تعديل" (أيقونة القلم) للخاصية المطلوبة
```

### الخطوة 3️⃣: إضافة القيم
```
1. في الجانب الأيسر من الصفحة → قسم "قيم الخاصية"
2. اضغط زر "إضافة قيمة" (الأخضر)
3. نافذة منبثقة ستظهر
```

### الخطوة 4️⃣: ملء بيانات القيمة
في النافذة المنبثقة:

**أ) القيمة*** (إجباري)
```
أمثلة:
- لخاصية "عدد الضيوف": اكتب "50-100 شخص"
- لخاصية "المدينة": اكتب "الرياض"
- لخاصية "نوع القاعة": اكتب "قاعة داخلية"
```

**ب) Slug (اختياري)**
```
- اتركه فارغاً → سيتم توليده تلقائياً
- أو اكتب slug بالإنجليزية: "50-100", "riyadh", "indoor-hall"
```

**ج) الترتيب (اختياري)**
```
- رقم الترتيب (الأقل رقماً يظهر أولاً)
- مثال: 0, 1, 2, 3...
```

**د) نشط ✓**
```
- ضع علامة ✓ لتفعيل القيمة
- أزل العلامة لإخفائها
```

### الخطوة 5️⃣: حفظ
```
اضغط زر "إضافة" (الأخضر)
```

---

## 📋 الطريقة الثانية: من صفحة تعديل الخدمة

### عند إنشاء/تعديل خدمة متغيرة:
```
1. افتح صفحة تعديل الخدمة: /admin/services/{id}/edit
2. اختر نوع الخدمة: "خدمة متغيرة"
3. في تبويب "التسعير والخيارات"
4. ستظهر شبكة الخصائص المتاحة
5. لكل خاصية ترى عدد القيم الموجودة
6. للإضافة → اذهب إلى صفحة الخاصية من زر "تعديل"
```

## 💡 أمثلة عملية

### مثال 1: خاصية "عدد الضيوف"

**الخطوات:**
1. اذهب إلى `/admin/attributes`
2. اضغط "تعديل" على خاصية "عدد الضيوف"
3. اضغط "إضافة قيمة"
4. أضف القيم التالية:

| القيمة | الترتيب | نشط |
|--------|---------|------|
| أقل من 50 شخص | 0 | ✓ |
| 50-100 شخص | 1 | ✓ |
| 100-200 شخص | 2 | ✓ |
| 200-300 شخص | 3 | ✓ |
| أكثر من 300 شخص | 4 | ✓ |

### مثال 2: خاصية "المدينة"

| القيمة | الترتيب | نشط |
|--------|---------|------|
| الرياض | 0 | ✓ |
| جدة | 1 | ✓ |
| الدمام | 2 | ✓ |
| مكة المكرمة | 3 | ✓ |
| المدينة المنورة | 4 | ✓ |

### مثال 3: خاصية "نوع القاعة"

| القيمة | الترتيب | نشط |
|--------|---------|------|
| قاعة داخلية | 0 | ✓ |
| قاعة خارجية | 1 | ✓ |
| فندق 5 نجوم | 2 | ✓ |
| منتجع سياحي | 3 | ✓ |

## ⚙️ إدارة القيم

### ✏️ تعديل قيمة موجودة:
1. في صفحة تعديل الخاصية
2. اضغط زر "تعديل" (أيقونة القلم) بجانب القيمة
3. عدّل البيانات
4. اضغط "حفظ التعديلات"

### 🗑️ حذف قيمة:
1. اضغط زر "حذف" (أيقونة سلة المهملات)
2. أكد الحذف

### ⚠️ تعطيل قيمة مؤقتاً:
1. افتح "تعديل" للقيمة
2. أزل علامة ✓ من "نشط"
3. احفظ → لن تظهر للجمهور لكن تبقى في النظام

## 📊 بعد إضافة القيم

### التحقق من النجاح:
✅ ستظهر القيم في جدول "قيم الخاصية"
✅ عند إنشاء/تعديل خدمة متغيرة → ستظهر هذه القيم في شبكة الخصائص
✅ عند إدارة التنويعات → ستظهر في قوائم الاختيار
✅ في صفحة الخدمة للجمهور → ستظهر كخيارات للاختيار

## 🎯 الخطوة التالية: إنشاء التنويعات

بعد إضافة القيم لجميع الخصائص:

### الطريقة التلقائية (الأسرع):
1. اذهب إلى تعديل الخدمة
2. اضغط "إدارة التنويعات"
3. اضغط "توليد كل التنويعات تلقائياً"
4. سيتم إنشاء تنويعة لكل تركيبة ممكنة

**مثال:** 
- إذا كانت خاصية "عدد الضيوف" لها 5 قيم
- وخاصية "المدينة" لها 5 قيم
- **النتيجة:** 5 × 5 = 25 تنويعة تلقائياً! 🎉

### الطريقة اليدوية:
1. اضغط "إضافة تنويعة يدوياً"
2. اختر قيمة من كل خاصية
3. حدد السعر
4. احفظ

## 💡 نصائح هامة

### ✅ أفضل الممارسات:
- **رتب القيم منطقياً** (من الأصغر للأكبر، من الأرخص للأغلى)
- **استخدم أسماء واضحة** (50-100 شخص أفضل من "متوسط")
- **لا تكثر من الخيارات** (5-10 خيارات كافية لكل خاصية)
- **فعّل فقط القيم المتاحة** (عطّل القيم غير المتوفرة حالياً)

### ⚠️ تجنب:
- ❌ تكرار نفس القيمة بأسماء مختلفة
- ❌ إضافة قيم غير واقعية أو غير متوفرة
- ❌ ترك الترتيب عشوائي (سيربك العملاء)

## 🐛 حل المشاكل

### المشكلة: لا أرى الخصائص في صفحة الخدمة
**الحل:**
1. تأكد من ربط الخاصية بالخدمة في صفحة تعديل الخدمة
2. تأكد من تفعيل الخاصية (is_active = ✓)
3. تأكد من وجود قيم نشطة للخاصية

### المشكلة: القيم لا تظهر في التنويعات
**الحل:**
1. تأكد من أن القيم مفعلة (is_active = ✓)
2. امسح الكاش: `php artisan optimize:clear`
3. أعد تحميل الصفحة

### المشكلة: خطأ عند الحفظ
**الحل:**
1. تأكد من ملء حقل "القيمة" (إجباري)
2. تحقق من أن الـ Slug فريد (أو اتركه فارغاً)
3. راجع سجل الأخطاء: `tail storage/logs/laravel.log`
