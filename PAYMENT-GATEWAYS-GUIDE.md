## 📄 ماذا تحتاج من شهادة الـ IBAN؟

### المعلومات المطلوبة:

عند التسجيل في أي بوابة دفع، سيطلبون منك:

1. **رقم الـ IBAN**
   - مثال: `SA1234567890123456789012`
   
2. **اسم الحساب البنكي**
   - يجب أن يطابق اسم السجل التجاري

3. **اسم البنك**
   - مثال: الراجحي، الأهلي، الرياض، إلخ

4. **نسخة من شهادة IBAN** (PDF أو صورة)
   - يجب أن تكون واضحة
   - حديثة (خلال 3 أشهر)

5. **السويفت كود (SWIFT Code)** - اختياري
   - للمعاملات الدولية

## 📋 خطوات التسجيل في Moyasar

### 1️⃣ إنشاء حساب:
- اذهب إلى: https://moyasar.com
- اضغط "التسجيل"
- أدخل بياناتك:
  - اسم الشركة: **Your Events**
  - البريد: `sales@yourevents.sa`
  - رقم الجوال
  - نوع النشاط: **خدمات الفعاليات والمناسبات**

### 2️⃣ رفع المستندات:

سيطلبون منك رفع:

**أ) السجل التجاري:**
- نسخة واضحة (PDF أو صورة)
- ساري المفعول

**ب) شهادة IBAN:**
- نسخة من الشهادة التي معك
- واضحة وحديثة

**ج) الهوية الوطنية:**
- صورة من وجهين البطاقة
- للمالك أو المفوض بالتوقيع

**د) معلومات الحساب البنكي:**
```
رقم الـ IBAN: SAxxxxxxxxxxxxxxxxxx
اسم صاحب الحساب: Your Events (أو حسب السجل)
اسم البنك: ____
```

### 3️⃣ مراجعة الحساب:
- سيراجع فريق Moyasar المستندات
- قد يتصلون بك للتأكيد
- المدة: 1-3 أيام عمل

### 4️⃣ الحصول على API Keys:
بعد الموافقة، ستحصل على:
```
Test API Key: pk_test_xxxxxxxxxxxxxx
Test Secret Key: sk_test_xxxxxxxxxxxxxx

Live API Key: pk_live_xxxxxxxxxxxxxx
```
