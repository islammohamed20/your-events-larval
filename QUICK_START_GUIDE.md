## 🎯 معلومات مهمة

### أرقام عروض الأسعار:
```
Format: QT-{YEAR}{NUMBER}
مثال:
QT-20250001
QT-20250002
QT-20250003
```

### حسابات الضريبة:
```php
المجموع الفرعي = مجموع (السعر × الكمية)
الضريبة = المجموع الفرعي × 0.15  // 15%
الإجمالي = المجموع الفرعي + الضريبة - الخصم
```

### حالات العرض:
```
pending   → قيد الانتظار (افتراضي)
approved  → موافق عليه
rejected  → مرفوض
completed → مكتمل
```

### الأذونات:
```
Cart: الجميع (زوار + مستخدمين)
Quotes: مستخدمين مسجلين فقط
Admin Quotes: مسؤولين فقط
```
