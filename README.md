# Your Events (Laravel)

منصة لإدارة الخدمات، العروض (Quotes)، والحجوزات، مبنية على Laravel.

## التثبيت السريع
- تثبيت الاعتمادات:
  - `composer install`
- إنشاء ملف البيئة:
  - `cp .env.example .env`
  - `php artisan key:generate`
- إعداد قاعدة البيانات داخل `.env` ثم:
  - `php artisan migrate`
- تشغيل محليًا:
  - `php artisan serve`

مزيد من التفاصيل في [docs/README.md](docs/README.md).

## ملاحظات أمنية مهمة
- لا ترفع ملف `.env` إلى GitHub (هو مُدرج ضمن `.gitignore`).
- أي مفاتيح دفع/SMTP يجب أن تبقى داخل `.env` أو Secrets في CI/CD.
