# 📄 شهادة الـ IBAN - ما تحتاجه بوابات الدفع

## 📋 المعلومات المطلوبة من شهادة الـ IBAN

عند التسجيل في **Moyasar** أو **HyperPay** أو أي بوابة دفع، سيطلبون منك:

### 1️⃣ **رقم الـ IBAN**
```
مثال: SA12 3456 7890 1234 5678 9012
```
- يتكون من 24 رقم (SA + 22 رقم)
- يبدأ بـ SA (السعودية)

---

### 2️⃣ **معلومات الحساب البنكي**

#### أ) اسم صاحب الحساب:
- يجب أن يطابق **اسم الشركة في السجل التجاري**
- مثال: `شركة يورافنتس للفعاليات` أو `Your Events Company`

#### ب) اسم البنك:
- مثال: مصرف الراجحي، البنك الأهلي، بنك الرياض، إلخ

#### ج) نوع الحساب:
- حساب جاري (Current Account) - الأفضل للشركات
- حساب توفير (Savings Account)

---

### 3️⃣ **نسخة من شهادة IBAN**

**المطلوب:**
- ✅ نسخة واضحة (PDF أو صورة عالية الجودة)
- ✅ شهادة رسمية من البنك
- ✅ حديثة (خلال 3 أشهر الأخيرة)
- ✅ تحتوي على:
  - شعار البنك
  - رقم الـ IBAN
  - اسم صاحب الحساب
  - تاريخ الإصدار
  - ختم البنك (إن وجد)

**كيف تحصل عليها؟**

**الطريقة 1 - تطبيق البنك (الأسرع):**
```
1. افتح تطبيق البنك (الراجحي، الأهلي، الرياض، إلخ)
2. اذهب إلى "حساباتي" أو "My Accounts"
3. اختر الحساب الجاري
4. اضغط "طلب شهادة IBAN" أو "IBAN Certificate"
5. ستصلك بالإيميل خلال دقائق (PDF)
```

**الطريقة 2 - موقع البنك:**
```
1. سجل دخول في موقع البنك
2. حساباتي → الحساب الجاري
3. طلب شهادة IBAN
4. تنزيل PDF
```

**الطريقة 3 - فرع البنك:**
```
1. اذهب لأقرب فرع
2. أطلب "شهادة إيبان للحساب الجاري"
3. استلمها مباشرة (مختومة)
```

---

### 4️⃣ **معلومات إضافية قد تُطلب**

#### أ) السويفت كود (SWIFT Code):
- للمعاملات الدولية
- مثال: `RJHISARI` (الراجحي)
- تجده في شهادة الـ IBAN

#### ب) رقم الحساب:
- 14-16 رقم
- جزء من الـ IBAN
- مثال: إذا كان IBAN: `SA1234567890123456789012`
  - رقم الحساب: `34567890123456789012`

#### ج) فرع البنك:
- اسم الفرع أو رقم الفرع
- مثال: "فرع الرياض الرئيسي"

## ✅ قائمة المستندات الكاملة لبوابة الدفع

عند التسجيل في **Moyasar** أو أي بوابة أخرى، ارفع:

### 1. شهادة IBAN ✅ (عندك)
   - PDF أو صورة واضحة
   - حديثة (خلال 3 أشهر)

### 2. السجل التجاري
   - نسخة واضحة
   - ساري المفعول
   - يحتوي على:
     - اسم الشركة
     - رقم السجل
     - تاريخ الانتهاء
     - النشاط التجاري

### 3. الهوية الوطنية
   - صورة من وجهين البطاقة
   - للمالك أو المفوض بالتوقيع
   - سارية المفعول

### 4. رخصة البلدية (اختياري)
   - إن وجدت
   - ساريه المفعول

### 5. عقد تأسيس الشركة (قد يُطلب)
   - نسخة من العقد الأصلي
   - موثق

## 🔍 كيف يتم التحقق من المستندات؟

### 1️⃣ التحقق التلقائي:
- يتحقق النظام من صحة رقم الـ IBAN
- يتحقق من تطابق الأسماء

### 2️⃣ المراجعة اليدوية:
- يراجع فريق الدعم المستندات
- قد يتصلون للتأكيد
- يتأكدون من النشاط التجاري

### 3️⃣ الموافقة:
- تستغرق 1-3 أيام عمل
- تصلك رسالة بريد إلكتروني
- يتم تفعيل API Keys


### ❌ الخطأ 3: شهادة قديمة
```
❌ شهادة من 6 أشهر

```

```
❌ صورة من الجوال غير واضحة
✅ PDF رسمي من البنك أو صورة عالية الجودة
```

## ✅ الخلاصة

### ما تحتاجه من شهادة IBAN:

1. ✅ **رقم الـ IBAN** (24 رقم)
2. ✅ **اسم صاحب الحساب** (يطابق السجل التجاري)
3. ✅ **اسم البنك**
4. ✅ **نسخة PDF من الشهادة** (واضحة، حديثة)

### خطوات الحصول عليها:

1. افتح تطبيق البنك
2. اطلب "شهادة IBAN"
3. انتظر دقائق (تصلك بالإيميل)
4. ارفعها عند التسجيل في Moyasar

### بعد رفع المستندات:

- الانتظار 1-3 أيام
- استلام API Keys
- البدء في قبول المدفوعات! 💳
