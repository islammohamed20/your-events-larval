# ميزة الشروط والأحكام - Terms and Conditions Feature

## 📋 نظرة عامة
تم إضافة نظام كامل لإدارة وعرض الشروط والأحكام الخاصة بالموقع.

## ✨ المميزات المضافة

### 1. **إدارة الشروط والأحكام من لوحة التحكم**
- حقل نصي كبير في صفحة الإعدادات (تبويب "المعلومات العامة")
- إمكانية كتابة شروط وأحكام مفصلة (حتى 50,000 حرف)
- حفظ تلقائي في ملف `storage/app/settings.json`
- مسح الـ cache تلقائياً بعد التحديث

### 2. **صفحة عرض احترافية**
- تصميم responsive يعمل على جميع الأجهزة
- تصميم احترافي بألوان الموقع
- عرض معلومات التواصل في حالة وجود استفسارات
- رسالة ودية عند عدم وجود محتوى
- أيقونات وتنسيق جذاب

### 3. **روابط في Footer**
- رابط في قسم "الروابط السريعة"
- رابط في أسفل الصفحة (Copyright section)
- تأثيرات hover جميلة على الروابط
- أيقونة مميزة للشروط والأحكام

## 🗂️ الملفات المعدلة/المضافة


## 🔧 التفاصيل التقنية

### 1. Database/Storage
- يتم حفظ البيانات في: `storage/app/settings.json`
- المفتاح: `terms_and_conditions`
- النوع: Text (حتى 50,000 حرف)

### 2. Validation Rules
```php
'terms_and_conditions' => 'nullable|string|max:50000'
```

### 3. Helper Function
```php
setting('terms_and_conditions')  // للحصول على الشروط والأحكام
```

### 4. Route
```
URL: /terms-and-conditions
Name: terms
Method: GET
```

## 📍 كيفية الوصول

### من لوحة التحكم (Admin):
1. اذهب إلى: `/admin/settings`
2. تبويب "المعلومات العامة"
3. ابحث عن حقل "الشروط والأحكام"
4. أدخل النص المطلوب
5. اضغط "حفظ الإعدادات"

### من الواجهة الأمامية (Frontend):
- **الرابط المباشر:** `/terms-and-conditions`
- **من Footer:** انقر على "الشروط والأحكام" في قسم الروابط السريعة
- **من أسفل الصفحة:** رابط في copyright section
