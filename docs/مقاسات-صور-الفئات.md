# مقاسات صور الفئات (Categories) في الصفحة الرئيسية

## 📐 المقاسات المطلوبة

### 1. **صورة الفئة (Category Image)**
- **الاستخدام**: تظهر كخلفية لبطاقة الفئة أو كصورة عائمة
- **المقاسات الموصى بها**: 
  - **العرض × الارتفاع**: `400×400 بكسل` (مربعة)
  - **نسبة العرض للارتفاع**: 1:1
  - **الحد الأدنى**: 300×300 بكسل
  - **الحد الأقصى**: 800×800 بكسل
- **الحجم المثالي**: 50-150 KB
- **التنسيق**: PNG (مع خلفية شفافة) أو JPG

#### كيف تُعرض الصورة:

**على الديسكتوب/التابلت:**
```
- عرض الصورة العائمة: 120px × 120px
- الموقع: أسفل يسار البطاقة
- تأثير: ظل منسدل (drop-shadow)
- object-fit: contain (تحافظ على النسبة الأصلية)
```

**على الموبايل:**
```
- عرض الصورة: 50px × 50px
- الموقع: في المنتصف أعلى النص
- تأثير: فلتر أبيض (brightness + invert)
- object-fit: contain
```

---

### 2. **الأيقونة (Icon)**
- **الاستخدام**: تظهر داخل صندوق مستدير في أعلى البطاقة
- **النوع**: Font Awesome Icon (مثل: `fas fa-camera`, `fas fa-music`)
- **المقاس عند العرض**:
  - **الديسكتوب**: صندوق 70×70 بكسل، أيقونة بحجم 2rem
  - **الموبايل**: صندوق 60×60 بكسل، أيقونة بحجم 1.6rem
- **اللون**: أبيض (#ffffff)
- **الخلفية**: شفافة مع blur effect

#### أمثلة للأيقونات المناسبة:
```html
<i class="fas fa-camera"></i>          <!-- تصوير -->
<i class="fas fa-music"></i>           <!-- موسيقى -->
<i class="fas fa-utensils"></i>        <!-- طعام -->
<i class="fas fa-palette"></i>         <!-- تصميم -->
<i class="fas fa-birthday-cake"></i>   <!-- حفلات -->
<i class="fas fa-theater-masks"></i>   <!-- ترفيه -->
<i class="fas fa-car"></i>             <!-- نقل -->
<i class="fas fa-hotel"></i>           <!-- ضيافة -->
```

## 📱 التفاصيل التقنية

### بطاقة الفئة - الديسكتوب (Desktop & Tablet)
```
- إجمالي ارتفاع البطاقة: ديناميكي (auto)
- عرض البطاقة: يتكيف مع الشبكة (3 أعمدة على lg, عمودين على md)
- قسم الهيدر: padding: 32px 24px 80px
- الصورة العائمة: 120×120 بكسل، موضع: bottom: 100px, left: 24px
- الأيقونة: 70×70 بكسل، border-radius: 18px
```

### بطاقة الفئة - الموبايل (Mobile)
```
- عرض البطاقة: 50% من الشاشة (عمودين)
- الحد الأدنى للارتفاع: 140px
- padding: 20px 16px
- الصورة/الأيقونة: 50×50 بكسل في المنتصف
- border-radius: 16px
```

## 🎨 إرشادات التصميم

### للصورة (Image):
1. ✅ **استخدم خلفية شفافة** (PNG) إذا كانت الصورة لها حواف غير منتظمة
2. ✅ **اجعل الصورة بسيطة وواضحة** - تجنب التفاصيل الدقيقة جداً
3. ✅ **استخدم ألوان متباينة** مع الخلفية الملونة (gradient)
4. ✅ **اختبر الصورة على خلفية داكنة** لأن الفئات تستخدم gradient أزرق/بنفسجي
5. ⚠️ **تجنب النصوص داخل الصورة** - لن تكون مقروءة بحجم 120px

### للأيقونة (Icon):
1. ✅ **اختر أيقونة معبرة** عن نوع الخدمة بوضوح
2. ✅ **استخدم Font Awesome 5 أو 6** (تأكد من التوافق)
3. ✅ **تجنب الأيقونات المعقدة** - البسيط أفضل
4. ✅ **اختبر الأيقونة على خلفية ملونة** لضمان الوضوح

## 📊 أمثلة عملية

### مثال 1: فئة "التصوير"
```
الصورة: camera-icon.png (400×400 بكسل، 80 KB)
الأيقونة: fas fa-camera
الخلفية: linear-gradient(135deg, #667eea 0%, #764ba2 100%)
```

### مثال 2: فئة "الطعام"
```
الصورة: food-catering.png (500×500 بكسل، 120 KB)
الأيقونة: fas fa-utensils
الخلفية: linear-gradient(135deg, #f093fb 0%, #f5576c 100%)
```

### مثال 3: فئة بدون صورة (أيقونة فقط)
```
الصورة: بدون
الأيقونة: fas fa-music
الخلفية: linear-gradient(135deg, #4facfe 0%, #00f2fe 100%)
ملاحظة: سيظهر pattern overlay تلقائياً
```

## 🔧 كيفية رفع الصور

### من لوحة الإدارة:
1. اذهب إلى **الفئات** في القائمة الجانبية
2. اختر الفئة أو أنشئ فئة جديدة
3. في حقل **الصورة**: ارفع الصورة (يُفضل PNG 400×400)
4. في حقل **الأيقونة**: أدخل كود Font Awesome (مثل: `fas fa-camera`)
5. احفظ التغييرات

### من الملفات:
```bash
# مسار رفع الصور
storage/app/public/categories/

# مثال
storage/app/public/categories/photography.png
```

## ✨ نصائح للحصول على أفضل نتيجة

1. **الأبعاد المثالية**: 400×400 بكسل (مربعة تماماً)
2. **الحجم الأمثل**: 50-100 KB (توازن بين الجودة والسرعة)
3. **التنسيق الأفضل**: PNG للصور ذات الخلفية الشفافة، JPG للصور العادية
4. **الأداء**: استخدم أدوات ضغط الصور (TinyPNG, ImageOptim) قبل الرفع
5. **الاختبار**: اختبر الصورة على أحجام شاشات مختلفة (ديسكتوب، تابلت، موبايل)

---

## 📝 ملاحظات مهمة

- الصورة **اختيارية** - إذا لم تُرفع صورة، ستظهر الأيقونة فقط مع خلفية gradient
- الصورة تُعرض بـ **object-fit: contain** - تحافظ على نسبة العرض للارتفاع الأصلية
- على الموبايل، الصورة تُعرض بـ **filter: brightness(0) invert(1)** - تتحول للأبيض
- التصميم **responsive** بالكامل - يتكيف تلقائياً مع جميع الشاشات
