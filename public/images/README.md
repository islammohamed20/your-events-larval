# دليل تنظيم صور الموقع

تم تنظيم صور الموقع في المجلدات التالية:

## 📁 المجلدات المتاحة:

### 🎭 `/vr/`
- **الغرض**: صور تقنية الواقع الافتراضي (VR)
- **أنواع الصور**: صور معدات VR، لقطات من التجارب الافتراضية، صور العروض التفاعلية
- **الأبعاد المقترحة**: 1920x1080 أو أعلى للجودة العالية
- **التنسيقات المدعومة**: JPG, PNG, WebP

### 🖼️ `/gallery/`
- **الغرض**: صور معرض الأحداث والفعاليات
- **أنواع الصور**: صور الأحداث السابقة، لقطات من الحفلات، صور العملاء
- **الأبعاد المقترحة**: 1200x800 للعرض الأمثل
- **التنسيقات المدعومة**: JPG, PNG, WebP

### 🛠️ `/services/`
- **الغرض**: صور الخدمات المقدمة
- **أنواع الصور**: صور المعدات، أنواع الخدمات، صور توضيحية للخدمات
- **الأبعاد المقترحة**: 800x600 أو 1:1 للبطاقات
- **التنسيقات المدعومة**: JPG, PNG, WebP, SVG (للأيقونات)

### 📦 `/packages/`
- **الغرض**: صور الباقات والعروض
- **أنواع الصور**: صور الباقات المختلفة، صور العروض الخاصة
- **الأبعاد المقترحة**: 600x400 أو 4:3
- **التنسيقات المدعومة**: JPG, PNG, WebP

### 🏷️ `/logo/`
- **الغرض**: شعارات الموقع
- **الملفات الحالية**: logo.png
- **الأبعاد المقترحة**: 200x60 للشعار الرئيسي
- **التنسيقات المدعومة**: PNG, SVG (مفضل)

## 📋 إرشادات عامة:

1. **أسماء الملفات**: استخدم أسماء وصفية باللغة الإنجليزية
2. **الحجم**: احرص على ضغط الصور لتحسين سرعة التحميل
3. **الجودة**: استخدم جودة عالية للصور المهمة
4. **التنسيق**: فضل WebP للويب، PNG للشفافية، JPG للصور العادية
5. **التسمية**: استخدم نمط واضح مثل: `service-wedding-planning.jpg`

## 🔗 الاستخدام في الكود:

```php
// مثال لاستخدام صور الخدمات
<img src="{{ asset('images/services/wedding-planning.jpg') }}" alt="تخطيط الأفراح">

// مثال لاستخدام صور الباقات
<img src="{{ asset('images/packages/premium-package.jpg') }}" alt="الباقة المميزة">

// مثال لاستخدام صور المعرض
<img src="{{ asset('images/gallery/event-2024-01.jpg') }}" alt="حدث يناير 2024">

// مثال لاستخدام صور VR
<img src="{{ asset('images/vr/vr-experience-1.jpg') }}" alt="تجربة الواقع الافتراضي">
```
